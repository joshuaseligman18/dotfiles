# dotfiles

## Required Homebrew Packages
### Formulae
* stow
* antigen
* starship
* neovim
* ripgrep
* fd
* tmux
* tpm

### Casks
* alacritty
* font-fira-code-nerd-font
* aerospace

## Setup
```bash
git clone --recursive https://github.com/joshuaseligman18/dotfiles ~/dotfiles
cd ~/dotfiles
stow .
```
